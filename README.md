# Smart Student Project Manager

A web-based system designed to manage student group projects, task collaboration, contribution tracking, and file sharing.

---

## Live Demo

https://smart-student-project-manager.vercel.app/

---

## Features

- User authentication and registration
- Group creation and management
- Kanban-style task board
- Task assignment and tracking
- Contribution tracking and gamification
- File upload and management
- Reports and analytics dashboard
- Role-based access control

## Installation

### Frontend

```bash
cd frontend
npm install
npm start
```

### Backend

```bash
cd Backend
pip install -r requirements.txt
python manage.py runserver
```
